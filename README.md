# puckstats

1. Download the [Puck Stats App](https://github.com/cheneryal/puckstats/releases/download/v1.0/Puck.Stats.exe)
2. Make sure Puck game is running and you are in a server
3. 
4. Run Puck Stats.exe
5. Click Draw Period ROI
6. Select the area where PERIOD, WARMUP, GOAL, REPLAY appears in your game and click Confirm ROI. V
7. Verify the correctly drawn ROI in the app window
8. Stats will automatically get logged when a game is played/spectated
