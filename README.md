# puckstats

1. Make sure Puck game is running and you are in a server
2. Run Puck Stats.exe
3. Click Draw Period ROI
4. Select the area where PERIOD, WARMUP, GOAL, REPLAY appears in your game and click Confirm ROI. V
5. Verify the correctly drawn ROI in the app window
6. Stats will automatically get logged when a game is played/spectated
